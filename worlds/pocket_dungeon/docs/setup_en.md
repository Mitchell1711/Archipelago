# Shovel Knight Pocket Dungeon MultiWorld Setup Guide

## Required Software
- [Archipelago](https://github.com/ArchipelagoMW/Archipelago/releases/latest)
- [Shovel Knight Pocket Dungeon on Steam](https://store.steampowered.com/app/1184760/Shovel_Knight_Pocket_Dungeon/)

## Installation Procedures

### Windows Setup
1. Download and install [Archipelago](https://github.com/ArchipelagoMW/Archipelago/releases/latest).
2. Download the latest [Shovel Knight Pocket Dungeon APWorld and the Archipelago mod from the releases page on this GitHub](https://github.com/Mitchell1711/Archipelago/releases/latest) and navigate to your Archipelago installation folder (by default this will be in ProgramData/Archipelago).
3. Place the Shovel Knight Pocket Dungeon APWorld into your custom_worlds folder.
4. Next, navigate to where your savedata is stored for Pocket Dungeon, by default this should be in "Users/{yourusername}/AppData/Roaming/Yacht Club Games/Shovel Knight Pocket Dungeon"
5. Add a new folder called "mods" if it's not yet present, then move the Archipelago mod into that folder and unzip it.

### Playing the game
If you've followed the setup instructions you can open the Archipelago Launcher and launch the Shovel Knight Pocket Dungeon client, if you had folder paths different from the default you can run the /savepath, /modpath and /gamepath commands to change them in the client. If everything is set up properly the Archipelago client will automatically boot up the game for you when you connect to a multiworld server!

### Where do I get a YAML file?
When you have the Shovel Knight Pocket Dungeon APWorld installed you can generate one by clicking "Generate Template Options" inside of the Archipelago Launcher, they are also provided with the latest release.
